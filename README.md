# 🎨 UIFlow - Assistente de Design UI/UX

UIFlow é um assistente de inteligência artificial feito com Streamlit e Groq, focado em ajudar iniciantes em Design UI/UX.

---

## 🚀 Funcionalidades

- Assistente de IA especializado em UI/UX
- Ajuda com layouts, cores e tipografia
- Sugestões de design para interfaces
- Interface simples e interativa
- Chat com histórico de mensagens

---

## 🛠️ Tecnologias

- Python
- Streamlit
- Groq API
